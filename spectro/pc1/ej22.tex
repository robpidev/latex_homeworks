Un electrón en un átomo de hidrógeno tiene un momento angular
$3.165\times10^{-34} Js$. ¿en qué orbita ($n$) se encuentra
el electrón?

\vspace{12pt}
\textit{SOLUCIÓN.} Utilizando la eq~\ref{eq:angular:momentum}

\begin{equation*}
    L = n\hbar \implies n = \frac{L}{\hbar}
\end{equation*}

remplazando con los valores dados

\begin{equation*}
    n \approx \frac{3.165\times10^{-34} Js}{1.054\times10^{-34}\;\mathrm{Js}} = 2.99
    \implies n = 3
\end{equation*}

\hfill $\blacksquare$
