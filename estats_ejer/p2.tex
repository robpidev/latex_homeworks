Determinar el momento magnético de un gas ideal de fermiones
dentro de un campo magnético uniforme $B$ en el 0 absoluto
de temperatura. En este caso la energía de un fermion es:
$p^2/2m\pm \mu_B B$ donde $\mu_B$ es el magneton de Bohr.

\sol Del problema la energía es

\begin{equation}
    \e = \frac{p^2}{2m} \pm \mu_B B
\end{equation}

como la temperatura es $T = 0$ y dado que la distribución de Fermi
está dada por

\begin{equation}
    \bk{n} = \frac{1}{e^{\beta(\mu - \e) } + 1}
\end{equation}

Para las condiciones del problema se tiene que $\mu = \e_f$
